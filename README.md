# Population Dynamics and Projections (PDP) 2026

This repository contains teaching materials and code for the practical sessions of the course Population Dynamics and Projections (PDP).

---------------------------------------------------------------------

REPOSITORY STRUCTURE

The repository is organised as follows:

PDP_2026/

    README.txt
        Repository description and instructions

    src/
        Scripts containing the code used in the practical sessions and slides

---------------------------------------------------------------------

SRC FOLDER

The src folder contains the scripts used during the course practicals. These scripts include:

• Simulation of demographic processes  
• Implementation of statistical models  
• Forecasting exercises  
• Slides  

The scripts are intended to be run sequentially during the practical sessions.

---------------------------------------------------------------------

REQUIREMENTS

To run the code in this repository you may need:

• R (recommended version 4.2 or higher)

Common R packages used for statistical modelling and visualisation may include:

tidyverse  
ggplot2  

You can install them using:

install.packages(c("tidyverse", "ggplot2"))

Additional packages may be required depending on the specific scripts and will be indicated within the code.

---------------------------------------------------------------------

PURPOSE OF THE REPOSITORY

The goal of this repository is to:

• Provide reproducible examples used during the course  
• Help students understand demographic forecasting methods  
• Illustrate the implementation of Bayesian models in demographic analysis  

---------------------------------------------------------------------

USAGE

1. Clone the repository:

git clone https://github.com/yourusername/PDP_2026.git

2. Navigate to the repository folder.

3. Run the scripts located in the src directory.
